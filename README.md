## About This Portfolio

This is a modern, executive-level portfolio website showcasing **10+ years of brand strategy, corporate communications, and digital growth expertise** across:
- Digital/Technology
- Real Estate & PropTech  
- Entertainment & Media

### Key Features

| Feature | Description |
|---------|-------------|
| **Hero Section** | Clear value proposition positioning as Senior Brand & Communications Leader |
| **Brand Impact Dashboard** | Animated metrics showing 42% engagement growth, 45% organic traffic, 40% brand recognition |
| **Case Studies** | 3 detailed projects with Challenge → Insight → Strategy → Execution → Impact framework |
| **Brand Growth Framework** | 5-step methodology: Discovery, Strategy, Creation, Activation, Optimization |
| **Organizations I Help** | 6 target company types with specific pain points and solutions |
| **Executive CTA** | Strong closing targeting Head of Brand, Brand Strategy Director, Corporate Communications Lead roles |

### Design System

- **Colors:** Deep purple (#1a0b2e, #2d1b4e) with deep pink accents (#e91e63, #f06292)
- **Typography:** Inter (sans-serif) + Playfair Display (serif)
- **Effects:** Glass morphism, gradient text, scroll animations, parallax blur orbs
- **Framework:** Tailwind CSS (via CDN)
